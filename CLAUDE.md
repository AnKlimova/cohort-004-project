# Git workflow

This repo has two remotes:

- `origin` — upstream repo (`ai-hero-dev/cohort-004-project`). Pull updates from here.
- `fork` — personal fork (`AnKlimova/cohort-004-project`). Push all changes here.

```bash
git pull origin main   # get latest upstream changes
git push fork main     # publish your changes
```

Never push to `origin`.
